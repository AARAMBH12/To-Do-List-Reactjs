# ReactJS To-Do List

A simple and interactive **To-Do List** application built with **ReactJS** and **Vite**.  
Users can add, complete, and delete tasks with a modern interface.

---

## 1. Features

1. Add new tasks  
2. Mark tasks as completed  
3. Delete tasks  
4. Responsive design for desktop and mobile  

---

## 2. Tech Stack

1. **Frontend:** ReactJS, JSX, CSS  
2. **Build Tool:** Vite  
3. **State Management:** React Hooks  
4. **Version Control:** Git & GitHub
